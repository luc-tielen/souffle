Detailed Description

template<unsigned arity, typename... Indices>
class souffle::ram::detail::AutoRelation< arity, Indices >

The relation implementing the type if relation utilizing a mixture of different types of data structures for its representation.

The most generic implementation of a relation supporting arbitrary arities > 0 and consistent lists of indices.

The actual data structures are implementation dependent.

Template Parameters
arity.. the arity of the resulting relation
Indices.. the indices to be maintained on top
